# Django School Diary Scripts

Набор вспомогательных функций для работы с дневником учеников в Django-проекте.

Скрипты позволяют:

исправлять плохие оценки
удалять замечания
добавлять похвалу ученикам


## Установка и подготовка

Склонируйте репозиторий и перейдите в папку проекта
Активируйте виртуальное окружение
Примените миграции:

```bach
python manage.py migrate
```

Запустите Django shell:
```bach
python manage.py shell
```

Импортируйте функции:
```bach
from datacenter.scripts import fix_marks, remove_chastisements, create_commendation
```


## функции

**get_schoolkid(name)**

Проверяет на правильно ли записано имя и находит его полное имя

```bach
get_schoolkid("Фролов Иван")
```

Возможные результаты:

"Ученик не найден"
"Найдено несколько учеников"

**fix_marks(name)**

Исправляет все плохие оценки (2 и 3) на 5 у указанного ученика.

Пример:
``` bach
fix_marks("Фролов Иван")
```


**remove_chastisements(name)**

Удаляет все замечания ученика.

Пример:

```bach
remove_chastisements("Фролов Иван")
```



**create_commendation(name, subject)**

Добавляет похвалу ученику по указанному предмету.

Что делает функция:

1.выбирает случайную похвалу
2.находит урок по предмету
3.создаёт запись с датой урока

Пример:

```bach
create_commendation("Фролов Иван", "Музыка")
```

Возможные результаты:

"Урок не найден"


## Ограничения

Имя ученика должно совпадать с данными в базе
Название предмета должно быть указано точно
Если найдено несколько учеников — функция не выполнится
Если урок не найден — похвала не создаётся


## Рекомендации

Используйте точные имена учеников
Проверяйте доступные предметы:
Subject.objects.values_list('title', flat=True)

Для отладки проверяйте данные:
Schoolkid.objects.all()
